## 安装软件
### npm换源
推荐使用换源工具`nrm`
``` 
npm install -g nrm   
```
查看源
```
nrm ls
```
![源列表](https://blogimbois-1256390899.cos.ap-beijing.myqcloud.com/post/Snipaste_2021-06-06_00-05-20.png)
添加源
```
nrm add 源的名称  https:// 地址
```
删除源
```
nrm del 源的名字
```
切换源
```
nrm use 源的名字
```
测试源速度
```
nrm test
```
